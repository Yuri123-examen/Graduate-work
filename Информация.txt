В архиве под названием "Скрипты" находятся 4 файла с расширением .py
Для того чтобы посмотреть код достаточно изменить формат на .txt
___________________________________________________________________________________________________________________________________________
Скрипты Face_Trainer и Face_Recog запускаются без виртуального окружения на самой плате Raspberry Pi 3 Model B, 
со всеми раннее необходимыми установленным библиотеками,
подробное пошаговое описание (шаг 1, шаг 2, шаг 3) есть в самой ВКР со всеми пояснениями и фотографиями действий.
____________________________________________________________________________________________________________________________________________
Скрипт train_model.py служит для создания и обучения нейросетевой модели на стационарном ПК с ОС windows 10, все 
действия по установке необходимых компонентов есть в файле ВКР (необходимо виртуальное окружение)
Скрипт detect.py запускается на Raspberry, необходимо также иметь виртуальное окружение.
____________________________________________________________________________________________________________________________________________
Для просмотра результатов выполненных задач необходимо скачать архивы "Дипломная работа(Распознавание лиц).rar" и
"Дорожные знаки - результат.rar"
Все архивы скачиваются без пароля. В каждом архиве содержатся медиафайлы (фотографии и видео), подтверждающие выполнение поставленных задач.
____________________________________________________________________________________________________________________________________________
Для того чтобы протестировать на своей плате корректность работы, достаточно изменить некоторые строки внутри скриптов. Например: URL
если вы используете камеру телефона по  WiFi и приложение IP Webcam, а также строки которые обозначают путь к вашим файлам.
